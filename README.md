## Run Locally

Clone the project

```bash
  git clone https://github.com/MoisalexWeb/brainwave-landing-page.git
```

Go to the project directory

```bash
  cd brainwave-landing-page
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
